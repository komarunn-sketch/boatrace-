📋 展示コピー機能 導入ガイド

ボートレース日和の展示データをワンタップでコピー

🎯 できること

🌐
予想ツールの「展示情報を開く」をタップ
↓
📄
「直前情報」タブをタップして展示データを表示
↓
📋
右下の「展示コピー」ボタンをタップ
↓
✅
予想ツールに戻って貼り付け → 自動解析完了！

⚙️ 導入方法を選んでください

Userscripts
推奨・自動実行
ブックマークレット
アプリ不要
1Userscriptsアプリをインストール
App Storeで「Userscripts」を検索してインストール（無料）
📲 App Storeで開く
2Safariで拡張機能を有効化
設定 → Safari → 機能拡張 → Userscripts → オンにする
3スクリプトファイルを保存するフォルダを作成
「ファイル」アプリを開く → iCloud Drive内に「Userscripts」フォルダを新規作成
4Userscriptsアプリでフォルダを指定
Userscriptsアプリを開く → 「Change Userscripts Directory」をタップ → 手順3で作ったフォルダを選択
5スクリプトファイルをフォルダに入れる
下のボタンでスクリプトをダウンロード → 「ファイル」アプリで手順3のフォルダに移動
コピーしたら「ファイル」アプリで新規テキストファイルを作成し、内容を貼り付けてboatrace_tenji.user.jsという名前でUserscriptsフォルダに保存してください
6Safariで展示ページを開いて許可
展示ページを開く → アドレスバー左の「ぁあ」→「Userscripts」→ 許可する
右下に「📋 展示コピー」ボタンが表示されれば完了！
✅ 一度設定すればボートレース日和の展示ページを開くたびに自動でボタンが表示されます
1Safariで適当なページをブックマーク登録
任意のページで「共有」→「ブックマークを追加」→ 名前を「展示コピー」に変更して保存
2ブックマークのURLを書き換える
ブックマーク一覧を開く → 「展示コピー」を長押し → 「編集」→ URLを以下に書き換えて保存
javascript:(function(){var rowOrder=['進入','展示','周回','周り足','直線','ST','体重','調整重量','チルト'];var rows={};document.querySelectorAll('td.shusso_title.komoku_title').forEach(function(td){var label=td.textContent.trim();if(!rowOrder.includes(label))return;var tr=td.closest('tr');if(!tr)return;var cells=tr.querySelectorAll('td.font100.center');var values=Array.from(cells).map(function(c){return c.textContent.trim().replace(/[^\d.\-F]/g,'');}).filter(function(v){return v!=='';});if(values.length>0)rows[label]=values;});if(Object.keys(rows).length===0){alert('展示データが見つかりません。直前情報タブを開いてから実行してください。');return;}var lines=['展示情報','進入\t1\t2\t3\t4\t5\t6'];rowOrder.forEach(function(label){if(rows[label])lines.push(label+'\t'+rows[label].join('\t'));});var text=lines.join('\n');var ta=document.createElement('textarea');ta.value=text;ta.style='position:fixed;opacity:0;top:0;left:0;';document.body.appendChild(ta);ta.focus();ta.select();try{document.execCommand('copy');alert('✅ コピーしました！予想ツールに貼り付けてください');}catch(e){alert(text);}document.body.removeChild(ta);})();
3展示ページで実行
展示ページの「直前情報」タブを開く → ブックマーク一覧から「展示コピー」をタップ → 予想ツールに貼り付け
⚠️ iOSのSafariではブックマークレットが動作しない場合があります。その場合はUserscripts方式をお試しください。

❓ うまくいかない場合

「展示データが見つかりません」と表示される
「直前情報」タブをタップしてから再度「展示コピー」をタップしてください
ボタンが表示されない（Userscripts）
アドレスバー左の「ぁあ」→「Userscripts」→ 許可する、を確認してください
✅ コピーしました！
